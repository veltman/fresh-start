# Fresh Start

A full bootstrap script for a new installation of Ubuntu Desktop.

Installs:

* Node.js
* VLC
* Gimp
* PIP
* Ruby
* QGIS
* GRASS
* PostgreSQL
* TileMill
* Chromium
* AWS CLI tools
* virtualenv
* Sublime Text 2
* Local webserver

Configures:

* Git
* Sublime Text 2
* Privacy settings
* Projects folder
* Launcher icons
* Bookmarks
